# Nabholz Appraisal Group — Internal Tools

Internal staff tools for Nabholz Appraisal Group.

## Tools

- **[Appraisal Intake Wizard](appraisal-intake/)** — Guided intake tool for private appraisal orders. Walks staff through qualifying questions, handles all order types (estate, divorce, tax appeal, etc.), and produces a printable order summary with escalation flags.
